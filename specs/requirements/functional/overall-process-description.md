# Overall Process Description

See file `/docs/wiki/Home.md` for the overall process description of the 
Document Search and Highlighting (DSH) system. This document provides a high-level 
overview of the end-to-end workflow, from document upload to highlight generation and 
retrieval. It outlines the key components involved, their interactions, and the expected 
outcomes at each stage of the process. For detailed functional requirements and technical 
specifications, please refer to the respective documents in the `/specs/` directory.
