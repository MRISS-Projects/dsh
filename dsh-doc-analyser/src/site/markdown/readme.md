## Installation

### Pre-requisites

#### Java

A JDK/JRE 17+ is required to run this application or any of its components.

1. Download a JDK/JRE distribution from the
   [Oracle downloads page](https://www.oracle.com/java/technologies/downloads/).
2. Install it following the installer instructions. Note the installation folder.
3. Set the following environment variables:
   - **Windows**: Go to *Start → Control Panel → System → Advanced → Environment Variables*.
     - Add `JAVA_HOME` pointing to the JDK/JRE root folder.
     - Edit `PATH` and prepend `%JAVA_HOME%\bin`.
4. Verify the installation:

```
java -version
```

Expected output (version will differ):

```
java version "17.0.x"
Java(TM) SE Runtime Environment ...
Java HotSpot(TM) 64-Bit Server VM ...
```
